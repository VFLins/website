## O que é feito?

Os pontos de venda (PDV) instalados no seu estabelecimento enviam dados detalhados das suas vendas para o Governo, estes são dados fiscais, mas que têm muitas informações sobre a operação da sua empresa. O mesmo vale para as notas de compra que são lançadas no seu sistema para controle de custos e de entrada de estoque.

Em conjunto, as notas de venda representam tanto a entrada de caixa, quanto a saída de mercadorias, e vice-versa para as notas de compra. Estes dados podem ser usados para produzir as métricas necessárias para entender e identificar os pontos fracos e fortes da sua empresa, seja sobre:

1. o *estoque*,
2. as *finanças*, ou até
3. o *comportamento* dos seus clientes.

Os dados extraídos são usados para produzir um relatório indicando o desempenho da empresa nos segmentos citados acima, e pode incluir pontos de melhoria e recomendações para resolvê-los, breve análise econômica conjuntural, além dos dados brutos usados na análise.

## Como funciona?

A qualidade das informações depende da precisão dos dados fornecidos na operação do caixa, enquanto que a quantidade de informações e o nível de aprofundamento do relatório depende da necessidade do cliente:

| Benefício                                                                                                            | Essencial | Avançado | Completo |
| -------------------------------------------------------------------------------------------------------------------- | --------- | -------- | -------- |
| Análise transacional detalhada, envolvendo formas de pagamento, volume de vendas ao longo do tempo, e endividamento. | ✅         | ✅        | ✅        |
| Análise estratégica, adicionando conjuntura econômica regional à análise anterior.                                   |           | ✅        | ✅        |
| Disponibilização dos dados estruturados.                                                                             |           |          | ✅        |
